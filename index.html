Repositories, issues, pull requests, CI/CD, packages, and more — all on your own hardware.

GitHub Actions-compatible CI/CD

Write your workflow once — it runs on both MyPersonalGit and GitHub Actions.

Parallel Jobs Independent jobs run concurrently. Use needs: for dependencies.
Matrix Builds strategy.matrix fans out jobs across OS, version, and variable combos.
Conditional Steps if: failure(), if: always(), and more for flexible control flow.
Step Outputs Pass data between steps with $GITHUB_OUTPUT.
Manual Triggers workflow_dispatch with typed inputs shown in the UI.
Job Timeouts timeout-minutes enforced per job with clear failure messages.
Secrets AES-256 encrypted secrets injected as env vars at runtime.
Status Badges Embeddable SVG badges for build and commit status.

Quick Start

1

Run with Docker

docker run -d --name mypersonalgit \
  -p 8080:8080 -p 2222:2222 \
  -v mypersonalgit-repos:/repos \
  -v mypersonalgit-data:/data \
  -v /var/run/docker.sock:/var/run/docker.sock \
  fennch/mypersonalgit:latest
2

Sign in

Open http://localhost:8080 and log in with admin / admin. Change the password immediately.

3

Create & push

git clone http://localhost:8080/git/MyRepo.git
cd MyRepo && echo "# Hello" > README.md
git add . && git commit -m "first commit"
git push origin main
